# Myth vs Fact Quiz – Mental Health Edition

## Project Overview
The Myth vs Fact Quiz is a simple web-based quiz designed to educate users about common mental health myths and facts. The goal is to raise awareness, reduce stigma, and make learning interactive and engaging.

This project currently focuses on the front-end structure using HTML and Bootstrap, with placeholders for future JavaScript functionality.

---

##  Features
- Myth vs Fact quiz format
- Multiple quiz modes:
  - Casual Mode (no timer)
  - Speed Mode (5-second timer)
  - Hardcore Mode (3-second timer)
- Accessibility options:
  - Sound toggle
  - Keyboard mode
  - Reduced motion
- Results summary with:
  - Final score
  - Accuracy
  - Category breakdown
- Review missed questions
- Replay and share options

---

##  Technologies Used
- **HTML5** – Page structure
- **Bootstrap 5** – Styling and layout
- **Font Awesome** – Icons (future use)
